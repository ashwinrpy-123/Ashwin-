# Professional Personal Portfolio Website

A static personal portfolio website built using only HTML5 and CSS3.

## Features
- **No JavaScript**: Pure HTML/CSS implementation (except for PWA JSON).
- **Responsive**: Mobile-first design using Flexbox and Grid.
- **PWA Ready**: Includes manifest.json and offline fallback.
- **Clean Design**: Modern Blue & Gray color theme.

## Structure
- `index.html`: Home page
- `about.html`: About & Education
- `skills.html`: Skills showcase
- `projects.html`: Project portfolio
- `contact.html`: Contact form
- `css/style.css`: Main stylesheet

## Usage
Simply open `index.html` in any modern web browser.

## Customization
- Update text content in HTML files.
- Change colors in `css/style.css` root variables.
- Add your images to `assets/images`.
